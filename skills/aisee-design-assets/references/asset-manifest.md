# 素材清单与复核

## 素材类型

- `background`：页面背景、Hero 背景、模块背景
- `icon`：入口图标、功能图标、状态图标
- `illustration`：插画、场景图、人物/物体视觉
- `overlay`：光效、纹理、装饰层
- `layout-layer`：卡片底板、容器背景、视觉框架层
- `transparent`：需要 alpha 通道的独立 PNG
- `edit`：基于已有图修改后的版本

## 生成前必须明确

- 素材用途
- 依赖的参考图或 StyleSpec
- 尺寸或比例
- 是否透明
- 是否允许文字、Logo、UI 控件
- 输出目录
- 是否属于通用图标；通用图标优先记录库名和语义，不走生图

## 状态

- `candidate`：候选，未确认
- `usable`：可进入下游流程
- `needs-edit`：可保留但需要人工微调或继续编辑
- `rejected`：不建议使用
- `archived`：历史版本，仅用于追溯

## 复核清单

参考图：

- 平台比例是否正确
- 是否符合页面场景
- 是否有不该出现的文字、Logo、水印或 UI 控件
- 是否能作为视觉方向，不误导为最终稿

设计规范：

- 是否能追溯来源
- 是否覆盖色彩、构图、图形语言、素材需求和禁用项
- 是否足以驱动下一轮生成
- 是否过度抽象或含糊

素材：

- 是否符合用途
- 是否与参考图或 StyleSpec 一致
- 图标是否优先来自图标库，语义和风格是否一致
- 透明边缘是否干净
- 是否存在背景残留、绿边、毛边或错误阴影
- 是否需要重做或人工微调

## Manifest 字段

```markdown
| ID | 文件 | 类型 | 用途 | 来源 | 透明 | 尺寸 | 状态 | 备注 |
|----|------|------|------|------|------|------|------|------|
```

图标素材可追加语义字段：

```markdown
| 语义 | 推荐图标 | 备选 | 尺寸 | 风格 | 用途 | 状态 |
|------|----------|------|------|------|------|------|
```
