# StyleSpec 提取 Prompt 模板

```text
任务类型：从参考图提取结构化设计规范。
输入参考图：<path>
适用平台：<mobile | mini-program | h5 | pc | desktop | multi>
输出格式：Markdown / JSON / both

请提取以下内容：
1. 视觉定位：关键词、整体气质、适合场景、不适合方向。
2. 色彩：主色、辅助色、背景色、强调色、禁用或慎用颜色。
3. 构图与层级：画布比例、视觉重心、留白、背景层、主体层、装饰层。
4. 图形语言：形状、线条、光影、材质、图标风格、插画风格。
5. 素材需求：背景、图标、插画、装饰层、布局视觉层、透明素材。
6. 禁止项：后续生成时必须避免的文字、Logo、水印、错误元素。
7. 下游生成提示：基于该规范生成新图时应保留、强调和避免什么。

如果参考图中某些细节不清楚，请标注为“不确定”，不要编造。
```
