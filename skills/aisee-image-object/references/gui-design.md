# PySide6 GUI 设计

GUI 第一版使用中文界面，专业术语可保留英文。

## 布局

- 左侧：原图画布和工具。
- 右侧：处理结果预览，尺寸尽量与左侧一致。
- 底部或侧栏：导出设置、素材列表、日志状态。

## 左侧原图画布

能力：

- 显示 `source.png`
- 显示 mask overlay
- 支持前景点、背景点、BBox 选择
- 支持缩放、平移、适应窗口

## 右侧结果预览

预览模式：

- 原图
- Mask
- Alpha
- Cutout
- 裁切结果

背景检查：

- 透明棋盘格
- 白底
- 黑底
- 自定义纯色

细节查看：

- 100%
- 200%
- 400%
- 适应窗口

## 导出设置

用户可控制：

- 是否透明
- 背景颜色或背景图
- 圆角大小
- Padding
- 裁切模式：按对象 BBox、原画布、正方形
- 格式：PNG、WebP、JPG

## 不放入 MVP 的能力

- GUI 不直接调用 Image Gen 深度优化。
- GUI 可以展示 `enhanced/` 下已有候选结果。
- 深度优化由对话或 Codex CLI 基于当前图片项目发起。
- GUI 中的框选区域、透明素材、抹除结果和导出结果都可以作为深度优化输入；用户在对话中提出优化要求时，由 agent 从 `source.json` 摘取相关过程文件生成轻量 handoff。
- GUI 不需要把“交给大模型优化”做成主流程按钮，避免用户误以为所有素材都必须走 Image Gen；它只负责把过程文件、scope 和备注保存清楚。
